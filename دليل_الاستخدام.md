## ✨ الوظائف المتاحة

### 📊 للمسؤول (Admin):

#### 1️⃣ إدارة العمارات:
- ✅ **عرض جميع العمارات** - اضغط على "العمارات" من القائمة
- ✅ **إضافة عمارة جديدة** - زر "إضافة عمارة"
  - الاسم: مثل "العمارة الثالثة"
  - العنوان: مثل "شارع الجامعة"
  - المدينة: مثل "الرياض"
  - عدد الوحدات: مثل "12"
- ✅ **تعديل عمارة** - اضغط على "تعديل" بجانب أي عمارة
- ✅ **حذف عمارة** - اضغط على "حذف" (سيطلب تأكيد)

#### 2️⃣ إدارة المستخدمين:
- ✅ **عرض جميع المستخدمين** - قسم "المستخدمين"
- ✅ **تفاصيل أي مستخدم** - اضغط على اسم المستخدم

#### 3️⃣ إدارة الوحدات:
- ✅ **إضافة وحدات للعمارات**
- ✅ **تعيين مستخدمين للوحدات**

---

### 👤 للمستخدم العادي:

- ✅ **عرض العمارات المتاحة**
- ✅ **عرض تفاصيل وحدته السكنية**
- ✅ **تحديث معلوماته الشخصية**

## 🧪 اختبار الوظائف خطوة بخطوة

### خطوة 1: تسجيل الدخول كمسؤول
1. افتح http://localhost:3000
2. أدخل: `admin@sakan.local` / `password`
3. اضغط "دخول"
4. ستظهر لوحة التحكم

### خطوة 2: عرض العمارات الحالية
1. من القائمة، اختر "العمارات"
2. سترى 2 عمارة:
   - العمارة الأولى - شارع النيل
   - العمارة الثانية - شارع التحرير

### خطوة 3: إضافة عمارة جديدة
1. اضغط زر "إضافة عمارة"
2. املأ البيانات:
   ```
   الاسم: العمارة الثالثة
   العنوان: شارع الملك فهد
   المدينة: جدة
   عدد الوحدات: 15
   ```
3. اضغط "حفظ"
4. ستظهر العمارة الجديدة في القائمة

### خطوة 4: تعديل عمارة
1. اضغط "تعديل" بجانب أي عمارة
2. غيّر البيانات
3. اضغط "حفظ التعديلات"

### خطوة 5: عرض المستخدمين
1. اختر "المستخدمين" من القائمة
2. سترى قائمة المستخدمين المسجلين

### خطوة 6: تجربة حساب مستخدم عادي
1. اضغط "تسجيل الخروج"
2. سجّل الدخول بـ: `user@sakan.local` / `password`
3. لاحظ الفرق في الصلاحيات

## 🔍 اختبار API مباشرة

### في المتصفح:
1. اضغط **F12** لفتح Developer Console
2. بعد تسجيل الدخول، ستجد Token في Console
3. يمكنك استخدامه لاختبار API

### مثال: الحصول على العمارات
افتح Console واكتب:
```javascript
fetch('http://localhost:5000/api/buildings', {
  headers: {
    'Authorization': 'Bearer ' + localStorage.getItem('token')
  }
})
.then(r => r.json())
.then(data => console.log(data));
```

### مثال: إضافة عمارة جديدة
```javascript
fetch('http://localhost:5000/api/buildings', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer ' + localStorage.getItem('token'),
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'العمارة الرابعة',
    address: 'شارع العروبة',
    city: 'الدمام',
    units_count: 20
  })
})
.then(r => r.json())
.then(data => console.log(data));
```

## 📡 قائمة API Endpoints الكاملة

### المصادقة
```
POST /api/auth/login       - تسجيل الدخول
POST /api/auth/register    - تسجيل جديد
```

### العمارات
```
GET    /api/buildings          - قائمة العمارات (للجميع)
POST   /api/buildings          - إضافة عمارة (Admin فقط)
GET    /api/buildings/:id      - تفاصيل عمارة (للجميع)
PUT    /api/buildings/:id      - تحديث عمارة (Admin فقط)
DELETE /api/buildings/:id      - حذف عمارة (Admin فقط)
```

### المستخدمين
```
GET    /api/users              - قائمة المستخدمين (مع Token)
GET    /api/users/:id          - تفاصيل مستخدم (مع Token)
PUT    /api/users/:id          - تحديث مستخدم (مع Token)
```

## 🐛 استكشاف الأخطاء

### المشكلة: لا يعمل تسجيل الدخول
**الحل:**
1. تأكد من Backend يعمل (البورت 5000)
2. افتح Developer Console (F12)
3. ابحث عن رسائل الخطأ
4. تأكد من إدخال البريد وكلمة المرور بشكل صحيح

### المشكلة: لا تظهر البيانات
**الحل:**
1. تحقق من Token في Console
2. تأكد من أن الـ API يعمل: http://localhost:5000/api/health
3. أعد تسجيل الدخول

### المشكلة: خطأ في الصلاحيات
**الحل:**
1. تأكد من أنك مسجل دخول كمسؤول (admin)
2. بعض الوظائف للمسؤول فقط (إضافة، تعديل، حذف)
